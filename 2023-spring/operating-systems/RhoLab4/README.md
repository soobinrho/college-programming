# Threads Deadlock Detection Program
Input a resource graph and check if it contains a deadlock or not.

<br>

***How to Compile and Run***
```bash
# Compile
g++ -o _ RhoLab4.cpp

# Run
./_

# How to use the example input using I/O redirectoin:
./_ < RhoLab4_inputExample.txt
```
